CDPilot

Click the toolbar button on a tab to attach / detach.

Getting started

If you see a red ! badge on the extension icon, the relay server is not reachable. Start the CDP relay server on this machine, then click the toolbar button again.

See the project README for setup instructions.

Relay port

Default: 18792. Extension connects to: http://127.0.0.1:<port>/. Only change this if your relay server uses a different port.

Auth Token

If your relay server requires authentication, enter the token here. It will be sent as a query parameter when connecting via WebSocket.

Recommended Chrome startup flags

For best anti-detection results, launch Chrome with the following flag:

--disable-blink-features=AutomationControlled

This prevents websites from detecting automation via navigator.webdriver and related Blink-level flags. Add this flag to your Chrome shortcut or startup script.
